# GBB
Gemini Build Builder (GBB) 是一个帮助将 Gemini ["Build your ideas with Gemini"](https://aistudio.google.com/apps) 功能生成的代码 zip 一键打包为 html+js+css 的小工具。

## 前置要求
[Node.js & npm](https://nodejs.org/): 建议安装 LTS 版本。

## 使用方法
将 zip 拖到 gbb.bat 上，或者
```powershell
# WIN
gbb path/to/source.zip
```
```sh
# Linux/MacOS (not stable)
bash gbb.sh /path/to/source.zip
```
